COURSE DESCRIPTION:

    This course will cover the basics of MFC Programming for those who understand C/C++ from the console application point of view.  An Introduction to Windows programming with and without the AppWizard (exe). This course is a preliminary course to Visual C++ Desktop Applications course for Microsoft Certification. Learn to create Windows programs using the Windows API's (Application Programming Interface). Review 32 bit multithreaded application design and the basics of ActiveX. This course will cover reading and writing to files to an introduction to Visual C++ and database programming.

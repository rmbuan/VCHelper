COURSE OUTCOMES:

  1. Write Win32 Applications to create Windows business applications, using Microsoft Foundation Classes.
  2. Use the graphical controls and their attributes.
  3. Create AppWizard(exe) multithreaded business applications, using the Class Wizard.
  4. Access the API functions of the Windows environment.
  5. Build Windows applications that read and write to files.
  6. Build Windows application that interact with local databases, as well as SQL Server databases.
  7. Create Dynamic Link Libraries and how to use them.
  8. Understand the creation of ActiveX Controls.
